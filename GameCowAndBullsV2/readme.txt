Cows And Bulls version 2
Play the “cows and bulls” game. The game works like this:
Randomly generate a 4-digit number.
User guess a 4-digit number.
For every digit that the user guessed correctly in the correct place, they have a “cow”.
For every digit the user guessed wrong but before they answer correctly for that place is a “bull.”
Every time the user makes a guess, and know how many “cows” and “bulls” they have.
Once the user guesses the correct number, the game is over. 
EXAMPLE
Say the number generated by the computer is 1038.
An example interaction could look like this:

Welcome to the Cows and Bulls Game!
----------------------------------
Enter a number:
>>> 1234
2 cows, 0 bulls
----------------------------------
>>> 1256
1 cow, 1 bull
...
Until the user guesses the number.
Every time user can exit from game
Just type exit or ctr + c
