This program check strongest of the provided password
and output strongest number and levels(weak, middle, strong)
STRONGEST NUMBER
This is the number that decides the password level
Every digit strongest added by 1
When password use numbers it added by 5
When password use uppercase letters it added by 10
When password use lowercase letters it added by 5
When password use symbols it added by 10

When you want exit just type exit
